## Contexto

Este projeto implementa um grafo não direcionado em linguagem C utilizando matriz de adjacência. O objetivo é aplicar conceitos de Estruturas de Dados e Teoria dos Grafos por meio de uma interface simples via terminal.

O sistema permite:

1. criar conexões entre vértices;
2. adicionar e remover arestas;
3. visualizar a matriz de adjacência;
4. calcular o grau dos vértices;
5. listar adjacências.

## Resultado

Ao final da execução, o usuário pode visualizar e analisar a estrutura do grafo, suas conexões e propriedades, facilitando o estudo e a compreensão dos conceitos fundamentais de grafos.
