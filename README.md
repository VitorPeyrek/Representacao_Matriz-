## Contexto

Grafos são estruturas de dados amplamente utilizadas para representar conexões entre elementos, sendo aplicados em redes sociais, sistemas de transporte, redes de computadores e diversos outros cenários.

Este projeto implementa um grafo não direcionado utilizando **Matriz de Adjacência** em linguagem C, permitindo a manipulação e análise das conexões entre vértices por meio de um menu interativo.

O sistema possibilita:

1. criar um grafo com quantidade definida de vértices;
2. adicionar e remover arestas;
3. visualizar a matriz de adjacência;
4. calcular o grau de cada vértice;
5. listar os vértices adjacentes;
6. reiniciar completamente o grafo.

## Resultado

Ao final da execução, o programa permite:

* visualizar a estrutura completa do grafo;
* analisar as conexões existentes entre os vértices;
* identificar o grau de cada vértice;
* consultar as adjacências de cada elemento;
* modificar dinamicamente as arestas;
* praticar conceitos fundamentais de Teoria dos Grafos e Estruturas de Dados.

## Tecnologias

* Linguagem C
* Estruturas de Dados
* Teoria dos Grafos
